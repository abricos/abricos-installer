Программная совместимость

Перед установкой платформы будут проведены проверки файлов и конфигурации сервера, чтобы удостовериться в том, что Abricos можно установить и запустить. Внимательно ознакомьтесь с результатами и не переходите к следующему шагу установки до тех пор, пока необходимые проверки не будут пройдены.

Версия и настройки PHP

Для установки Abricos требуется версия PHP не ниже 5.0.0. Если ниже написано Безопасный режим (Safe mode), то PHP работает в этом режиме. Это наложит некоторые ограничения на удалённое администрирование и аналогичные функции.

Версия PHP ≥ 5.0.0:
Параметр PHP register_globals отключён:
Abricos будет работать, даже если параметр register_globals включён. Но, по возможности, его рекомендуется отключить из соображений безопасности.
Параметр PHP allow_url_fopen включён:
Необязательно.
Функция PHP getimagesize() доступна:
Необходима. Для правильной работы Abricos необходимо наличие функции getimagesize.
Поддержка UTF-8 в PCRE:
Abricos не будет работать, если PHP собран без поддержки UTF-8 для расширения PCRE.

Проверка расширения mbstring

Необходимо. mbstring — это расширение PHP, обеспечивающее работу многобайтных строковых функций. Некоторые возможности mbstring несовместимы с Abricos и должны быть отключены.

Перегрузка функции:
Значение mbstring.func_overload должно быть установлено в 0 или в 4.
Прозрачное кодирование символов:
Значение mbstring.encoding_translation должно быть установлено в 0.
Кодировка входящих символов:
Значение mbstring.http_input должно быть установлено в pass.
Кодировка исходящих символов:
Значение mbstring.http_output должно быть установлено в pass.
Файлы и папки

Для правильной работы Abricos необходим доступ на чтение или запись определённых файлов и папок. Если вы увидите надпись «Не найден», то необходимо создать соответствующий файл или папку. А если увидите надпись «Нет прав на запись», то необходимо изменить права на файл или на папку, чтобы разрешить Abricos запись в них.
